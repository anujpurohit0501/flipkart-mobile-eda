Exploratory Data Analysis of Flipkart mobile dataset using Python to understand pricing, ratings, and feature trends.


# 📊 Flipkart Mobile Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis on Flipkart mobile phone dataset to understand trends in pricing, ratings, and mobile specifications.

---

## 🎯 Objective
To analyze mobile phone data and extract meaningful insights about price distribution, brand popularity, and feature patterns.

---

## 📂 Dataset Description
The dataset contains information about mobile phones including:
- Brand
- Model
- Selling Price
- Rating
- RAM
- Storage
- Original Price

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Key Analysis Performed
- Data cleaning (handling missing values & duplicates)
- Statistical summary of dataset
- Visualization of price distribution
- Brand-wise analysis
- Rating vs Price relationship
- Feature distribution (RAM, Storage)

---

## 📈 Key Insights
- Most phones are in mid-price range
- Samsung and Xiaomi are most common brands
- Higher price does not always mean higher rating
- 128GB storage is most popular

---
